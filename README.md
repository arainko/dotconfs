# dotconfs

## Installation

`ln -sf ~/repos/dotconfs/nvim ~/.config/nvim`
`ln -sf ~/repos/dotconfs/tmux/.tmux.conf ~/.tmux.conf`
`ln -sf ~/repos/dotconfs/bin/workspaces ~/.local/bin/workspaces`
`mkdir ~/.oh-my-zsh/completions`
`ln -sf ~/repos/dotconfs/zsh/completions/_workspaces ~/.oh-my-zsh/completions/_workspaces`
